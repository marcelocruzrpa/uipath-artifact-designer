# Security Policy

Help us improve the security of the UiPath Artifact Designer. We take security seriously and appreciate your help in keeping our community safe.

## Supported Versions

The following versions of the extension are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We use **GitHub Security Advisories Private Vulnerability Reporting** for coordinated disclosure. Please do **not** report security vulnerabilities via public GitHub issues or personal email.

### How to Report
1. Navigate to the [repository on GitHub](https://github.com/marcelocruzrpa/uipath-artifact-designer).
2. Click on the **Security** tab.
3. Click **Advisories** in the left sidebar.
4. Click **Report a vulnerability** to open the private reporting form.

### Our Commitment
This is a community-maintained project with a small maintainer team. When you report a vulnerability through the private channel we will, on a best-effort basis:
- **Acknowledge** receipt of your report, typically within **14 days**.
- **Update you** on the investigation and fix progress at meaningful milestones.
- **Aim to release** a fix and public advisory within **90 days** of the initial report, coordinated with you.

If you have not heard back after 21 days, please ping the advisory thread — reports occasionally fall through the cracks.

### Coordinated Disclosure
We ask that you do **not** publicly disclose the vulnerability or share details about it until a fix has been released and we have mutually agreed on a disclosure date. This helps protect users of the extension while a solution is being developed.

Thank you for your help in keeping this community project secure!
